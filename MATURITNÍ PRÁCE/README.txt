Aplikace na vyhledávání bydlení pro studenty.

1. Krok stáhněte si node.js - https://nodejs.org/en vždy nejnovější verzi pro nejlepší funkčnost.
2. Poté stáhněte skrz Command line do složky složku node_modules tímto příkazem - (npm install).
3. Jakmile stáhnete tuto složku tak napište do command line tento příkat - (npm run dev nebo npm start), tím spustíte stránku.
4. Když se vám ukáže odkaz localhost:XXXX, tak to pouze přepište do vašeho prohlížeče a stránka se vám zobrazí.
